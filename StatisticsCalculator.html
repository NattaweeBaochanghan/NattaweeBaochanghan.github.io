Statistics Calculator!

Write the web app to prompt accepting positive integers until the user enters a negative integer

  • If the entered text is not a positive integer number, then the web app will keep asking for a number until the valid positive integer is entered
  • If the entered text is a negative integer, then the web app will show the statistics(average, min, max) of all entered positive integers

Hint:

  • You need the implement function to check an entered text is a positive integer or not and whether an entered text is a negative integer or not
  • You need to implement function to compute average, min, and max of list of integers
